## 두 정수 사이의 합

**프로그래머스**에서 풀어본 ***첫 알고리즘 문제!***

<br>

- 두 정수 사이의 합을 구하시오
  - 정수 **a**와 **b**가 주어졌을 때 a와 b를 포함한 두 정수 사이의 모든 정수의 합을 구하는 문제

<br>

- 사용한 조건문
  - if __: __
      - ```if 조건이 참 혹은 거짓이라면: 다음 코드를 실행하시오.```

<br>

- 사용한 함수
  - range()
      - ```range([시작:int, default = 0,] 끝:int [, 스텝:int, default = 1])  # 끝은 포함하지 않는다.```
      - 예1) ```>>> range(5)```
        - ```range(0, 5)```
      - 예2) ```>>> list(range(5))```
        - ```[0, 1, 2, 3, 4]```
      - 예3) ```>>> range(1, 5)```
        - ```[1, 2, 3, 4]```
      - 예4) ```>>> list(range(0, 7, 2))```
        - ```[0, 2, 4, 6]```
